# Обход белых списков

Белые списки — список российских сервисов, которые работают при "отключении мобильного интернета".

Обход белых списков отличается у каждого пользователя, в зависимости от оператора и региона. То, что работает у одного, может не работать у другого, **универсального решения нет**!

Для подключения к конфигам используйте одно из этих приложений:
- [v2rayN](/network/vpns/v2rayn) (Windows, Linux, macOS)
- [v2rayNG](/network/vpns/v2rayng) (Android)
- Exclave (Android)
- V2box (iOS, Android)
- Karing (Android, iOS, Windows, Linux, macOS)
